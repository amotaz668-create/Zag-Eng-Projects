# Zag-Eng-Projects

# HTML Documentation

## Project 1 – Gas Cylinder Order Form

A simple HTML form designed to collect customer information for ordering or refilling gas cylinders.  
The form includes basic input fields such as text, email, phone number, checkboxes, dropdown menus, number fields, a textarea, and a submit button.

---

## Project 2 – Resume Page

A basic HTML resume layout that contains sections for personal information, career objective, educational qualifications, technical skills, and project experience.  
Tables are used for organizing education details and technical skills, and lists are used for projects.

---

## About This Repository

This repository includes two simple HTML projects that demonstrate the fundamentals of HTML structure, forms, tables, and page layout.  
It serves as basic practice for learning and applying core HTML concepts.
