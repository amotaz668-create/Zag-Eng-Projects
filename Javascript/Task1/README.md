# JavaScript Basics - Task 1

## Introduction

JavaScript is a high-level, interpreted programming language used to build dynamic and interactive web applications.

---

## Datatypes

### Primitive Datatypes

- Number
- String
- Boolean
- Undefined
- Null
- Symbol
- BigInt

### Non-Primitive Datatypes

- Object
- Array
- Function

---

## Variable Types

### var

- Function scoped
- Can be redeclared

### let

- Block scoped
- Can be reassigned

### const

- Block scoped
- Cannot be reassigned

---

## Number Object

The `Number()` function is used to convert values into numbers.

Example:

```js
Number("10"); // 10
```
